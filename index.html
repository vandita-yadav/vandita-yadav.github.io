Netflix & TV Shows Content Analyser

An interactive Streamlit-powered recommender system for 32,000+ movies and TV shows.

This project is a content-based recommender that achieves 85% relevance using genre and language insights[cite: 160]. It utilizes cosine similarity on combined features (genre, cast, director, description) to generate smart recommendations. Performance is highly optimized by accelerating memory handling with Joblib, reducing the 7.6 GB matrix load times by 40%[cite: 161].

Tech Stack

Python Pandas Scikit-learn Streamlit Joblib

Key Features

  • Content-based recommendations showing 5 similar titles.
  • Filtering for Top 20 titles globally by language or genre.
  • Interactive Streamlit dashboard with modular functions.
  • Advanced data preprocessing and text normalization.

Technical Workflow

  • Vectorized text using CountVectorizer.
  • Calculated cosine similarity for content matching.
  • Saved the massive 7.6 GB similarity matrix locally via Joblib.
  • Handled missing source data without breaking the UI.
